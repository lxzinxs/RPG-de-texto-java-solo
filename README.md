# [RPG DE TEXTO]

Este é um projeto pessoal idealizado com o objetivo de aprimorar minhas habilidades de programação, explorar novas tecnologias e enfrentar o desafio de construir algo do zero.

## 🚀 O Desafio (100% Manual)
Para maximizar o aprendizado, este projeto será desenvolvido **totalmente do zero e sem o uso de Inteligência Artificial (IA)**. Todo o processo de solução de problemas e lógica será baseado puramente em pesquisas em documentações oficiais, fóruns e artigos técnicos.

## 🔮 Planos Futuros
O projeto inicial serve como base para ideias mais ambiciosas. No futuro, pretendo expandi-lo para:
*   Integrar e transformá-lo em um **Bot de Discord**.
*   Desenvolver uma interface visual e design em **2D**.

---
*Projeto desenvolvido de forma solo, focado em aprendizado contínuo e evolução técnica.*
